# weather_check
基于深度学习的【天气识别】系统设计与实现~Python

## 一、介绍
天气识别系统，以Python作为主要编程语言，通过收集了4种常见的天气图像数据集（多云、雨天、晴天、日出），然后基于TensorFlow搭建卷积神经网络算法模型，通过多轮迭代训练，最后得到一个识别精度较高的h5格式模型文件，然后基于Django搭建Web网页端操作界面，实现用户上传一张天气图片识别其名称。
